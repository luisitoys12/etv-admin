Configuración de Firebase Firestore

1. Crea tu proyecto en Firebase

  1. Ve a Firebase Console y crea un nuevo proyecto.
  2. Agrega una app web desde el panel de inicio y copia tu configuración (apiKey, authDomain, projectId, etc).
  3. Activa Firestore Database desde el menú "Cloud Firestore".
Ejemplo de configuración en tu código: 
const firebaseConfig = {
  apiKey: "TU_API_KEY",
  authDomain: "TU_AUTH_DOMAIN",
  projectId: "TU_PROJECT_ID",
  // etc.
};

2. Estructura de colecciones y documentos

programacion (colección)

  • title (string): Título del programa
  • day (string): Día del programa (ej: "Lunes")
  • time (string): Hora (ej: "18:00")
  • description (string): Descripción
  • image (string): URL de la imagen
{
  "title": "Noticiero Central",
  "day": "Lunes",
  "time": "20:00",
  "description": "Noticias nacionales e internacionales.",
  "image": "https://tusitio.com/imagenes/noticiero.jpg"
}

top10 (colección) → config (documento)

  • image (string): URL de la imagen
  • playlistId (string): ID de la playlist de YouTube
{
  "image": "https://tusitio.com/imagenes/top10.jpg",
  "playlistId": "PLxxxxxxxxxxxxxxx"
}

mensajes (colección) → liveMessage (documento)

  • text (string): Mensaje en vivo
{
  "text": "¡Bienvenidos a nuestra programación en vivo!"
}

encuestas (colección) → currentPoll (documento)

  • question (string): Pregunta de la encuesta
  • options (array de objetos): Opciones de respuesta, cada una con campo text (string)
{
  "question": "¿Cuál es tu sección favorita?",
  "options": [
    { "text": "Noticias" },
    { "text": "Deportes" },
    { "text": "Cultura" }
  ]
}

3. Seguridad: Permisos mínimos de Firestore (lectura pública)

Para que la web pueda leer los datos de Firestore sin autenticación, configura las reglas de la base de datos así:
¡Atención! Esto solo permite lectura pública. Si necesitas seguridad extra, consulta la documentación oficial. 

service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
    }
  }
}
